## 3. 사용 기술

### 3.1 GUI 프레임워크
- **선택: Tkinter** ✅
  - Python 기본 내장
  - 크로스 플랫폼 지원 (Windows, macOS, Linux)
  - 간단한 구현과 유지보수
  - **다크 테마 디자인 시스템**
    - 현대적인 다크 색상 팔레트
    - 일관된 컴포넌트 스타일링
    - 향상된 사용자 경험

### 3.2 데이터 저장
- **설정 파일**: JSON 형식
- **임시 데이터**: SQLite
- **캐싱**: 메모리 캐시

### 3.3 외부 연동
- **Jira API**: REST API 사용
- **인증**: OAuth 2.0 또는 API Token

## 6. 플랫폼별 실행 방법

### Windows
```bash
# 방법 1: 배치 파일
run_gui.bat

# 방법 2: PowerShell
.\run_gui.ps1

# 방법 3: Python 직접 실행
python src\main.py
```

### macOS/Linux
```bash
# 방법 1: Shell 스크립트
./run_gui.sh

# 방법 2: Python 직접 실행
python3 src/main.py
```

### 설치 요구사항
- **Windows**: Python 3.8+ (tkinter 포함)
- **macOS**: Python 3.8+ (tkinter 포함)
- **Linux**: Python 3.8+ 및 python3-tk 패키지
